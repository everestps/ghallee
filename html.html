What is HTML?

Html stands for HyperText Markup Language. It is a scripting language used to create websites. Right click and select 'View Page Source' to see the HTML code for this website.

What is CSS?

CSS stands for Cascading Style Sheets. We use CSS to make websites look attractive. Without CSS websites look very bad. For an example visit this website. For more on how CSS beautfies websites look bellow

How do tags work?

Tags are individual elements of HTML. If you think about a website as an essay the HTML tags would be the words that make it up. The basic structure of a website is broken down into three parts contained inside special tags. The entire website is wrapped in <html> and </html> tags The top of that section of code is the head. It is contained inside <head> and </head> tags. Inside the head webdesigners put <title> and </title> tags and a <link> to their CSS document. Under the head is the <body>! The body contains everything you see on the website such as text photos, gifs, and links, in short content. You can see examples of how to create such content with HTML tags in the table to the right.
